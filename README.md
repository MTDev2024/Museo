## Stack

- React + Vite
- Tailwind CSS v4
- React Router DOM
- Framer Motion
- React Three Fiber (particules)
- API Art Institute of Chicago (gratuite, sans clé)

## Fonctionnalités

- Recherche d'artistes (Van Gogh, Monet, Caravaggio...)
- Galerie avec pagination (load more)
- Page détail avec parallaxe et zoom plein écran
- Système de favoris (localStorage)
- Thème sombre / clair
- Animations et transitions de page
